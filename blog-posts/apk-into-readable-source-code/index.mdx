**Prequisites**

* Java - should be able to run `java -version` in your terminal
* Bytecode Viewer

<Step number={1} title='Download the jar for Bytecode Viewer located at this link: [page](https://github.com/Konloch/bytecode-viewer/releases)'/>

<Step number={2} title='Use a site like apkpure.com to download the apk you want to look into'/>
For this guide, we'll be this [apk](https://m.apkpure.com/eme-hive-meet-chat-go-live/com.eastmeeteast.eme_android)

<Step number={3} title='Open Bytecode Viewer using the following command'/>
```bash
java -jar jda-1.2.0-with-deps
```

<Step number={4} title='You should see the application open up, with an option to drag a .jar file into the top left window'/>
  ![](/assets/images/blog-posts/apk-into-readable-source-code/1.png)

<Step number={5} title='Drag in the apk downloaded in step 2'/>

<Step number={6} title='You should see the source code after the application has processed the apk '/>
  ![](/assets/images/blog-posts/apk-into-readable-source-code/2.png)

<Step number={7} title='Looking through we can see several things about this application'/>

a. The usage of Firebase  
b. Kotlin based application  
c. Source code under `com.eastmeeteast`  

<Step number={8} title='Looking at one .class file, we can see the implementation. Example shown below is for `com.eastmeeteast.api.servicemodule.CommunitiesModule.class`'/>
 ![](/assets/images/blog-posts/apk-into-readable-source-code/3.png)

<Step number={9} title='You can change which disassembler/decompiler you use to view .class files under `View > Pane 1 OR Pane 2 OR Pane 3`'/>
As an example, I set pane 3 to be bytecode
![](/assets/images/blog-posts/apk-into-readable-source-code/4.png)

<Step number={10} title='Hit the Refresh button under Workspace window and you should see the pane changed to whatever you wanted to view the code in'/>
![](/assets/images/blog-posts/apk-into-readable-source-code/5.png)
