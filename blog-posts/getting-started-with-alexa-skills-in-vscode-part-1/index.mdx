There are plenty of tutorials out there for building Alexa Skills since it's introduction in 2014.

As of today, there seems to be better and better ways to get your environment setup for development quickly.
This tutorial is going to go over a relatively new feature in the Alexa dashboard to integrate with Visual Studio Code.

Let's get started!

<MarkdownStep number={1} title="Log onto the [Alexa Dashboard](https://developer.amazon.com/alexa/console/ask)."/>
   Create an account if you haven't already

<MarkdownStep number={2} title="Click `Create Skill` "/>
![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/1.png)

<MarkdownStep number={3} title="Enter any skill name and keep the rest as default"/>
![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/2.png)

<MarkdownStep number={4} title="Choose your template - you can go simple with Hello World or go with a more complicated template to see how those are built."/>
![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/3.png)

<MarkdownStep number={5} title="Continue and wait for your skill to be generated. This can take several minutes."/>

<MarkdownStep number={6} title="Once the skill is generated, it'll auto redirect you to the skill dashboard. "/>
   Lets click on the Code tab
![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/4.png)
<MarkdownStep number={7} title="Click the console icon and `Export to VS Code`"/>
   It'll prompt you to open Visual Studio Code. 
   ![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/5.png)

<MarkdownStep number={8} title="Allow access and then click the `Sign In` button on the tab that appears in VS Code. "/>
![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/6.png)

<MarkdownStep number={9} title="Go through the steps to allow access"/>
![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/7.png)

<MarkdownStep number={10} title="Head back to Visual Studio Code when you hit this screen"/>
![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/8.png)

<MarkdownStep number={11} title="Alexa Skills Kit (ASK) Toolkit should've been auto installed"/>

<MarkdownStep number={12} title="If you haven't been redirected correctly, go back to the skill dashboard and hit export to VS Code again"/>
It should ask which folder to download the skill to

<MarkdownStep number={13} title="This is the screen that should appear if everything has been done correctly."/>
![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/9.png)
