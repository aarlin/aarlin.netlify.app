## Tutorial

There are plenty of tutorials out there for building Alexa Skills since it's introduction in 2014.

As of today, there seems to be better and better ways to get your environment setup for development quickly.
This tutorial is going to go over a relatively new feature in the Alexa dashboard to integrate with Visual Studio Code.

Let's get started!

<Badge>1</Badge> Log onto the [Alexa Dashboard](https://developer.amazon.com/alexa/console/ask).
   Create an account if you haven't already

<Badge>2</Badge> Click `Create Skill` 

![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/1.png)

<Badge>3</Badge> Enter any skill name and keep the rest as default

![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/2.png)

<Badge>4</Badge> Choose your template - you can go simple with Hello World or go with a more complicated template to see how those are built.

![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/3.png)

<Badge>5</Badge> Continue and wait for your skill to be generated. This can take several minutes.

<Badge>6</Badge> Once the skill is generated, it'll auto redirect you to the skill dashboard. 
   Lets click on the Code tab

![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/4.png)

<Badge>7</Badge> Click the console icon and `Export to VS Code`
   It'll prompt you to open Visual Studio Code. 

   ![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/5.png)

<Badge>8</Badge> Allow access and then click the `Sign In` button on the tab that appears in VS Code. 

![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/6.png)

<Badge>9</Badge> Go through the steps to allow access

![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/7.png)

<Badge>10</Badge> Head back to Visual Studio Code when you hit this screen

![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/8.png)

<Badge>11</Badge> Alexa Skills Kit (ASK) Toolkit should've been auto installed

<Badge>12</Badge> If you haven't been redirected correctly, go back to the skill dashboard and hit export to VS Code again
It should ask which folder to download the skill to

<Badge>13</Badge> This is the screen that should appear if everything has been done correctly.

![](/assets/images/blog-posts/getting-started-with-alexa-skills-in-vscode-part-1/9.png)

<Badge>14</Badge> Time to start coding. There will be a part 2 for the features in the ASK toolkit
